# Revizorro Diff

Chrome-расширение для подсветки diff между итерациями код-ревью. Работает поверх существующей платформы ревью, добавляя визуальное сравнение версий кода прямо в интерфейсе.

## Возможности

- Построчная подсветка добавленных и удалённых строк (LCS diff)
- Бейджи на файлах: `новый` / `изменён` / `удалён`
- Статистика изменений на папках: `+5 ~3 -1`
- Виджет удалённых файлов внутри папок
- Переключение между итерациями для сравнения
- Автоматическое обновление при навигации (MutationObserver)

## Установка

1. Клонировать репозиторий
2. Открыть `chrome://extensions/`
3. Включить "Режим разработчика"
4. "Загрузить распакованное расширение" → выбрать папку `extension/`

## Структура

```
extension/
  manifest.json   — Manifest V3
  content.js      — Логика: парсинг данных, diff, DOM-инъекция
  content.css     — Стили подсветки и UI-компонентов
  popup.html/js   — Popup со статусом расширения
  icons/          — Иконки расширения
```

## Как работает

Расширение парсит данные из inline-скрипта страницы, строит индексы файлов и вычисляет LCS diff между выбранными итерациями. Результат инжектится в DOM как подсветка строк и бейджи. Observer отслеживает изменения DOM (открытие файлов, навигация) и применяет diff автоматически.
